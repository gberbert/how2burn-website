# Stack Tecnológica

## Visão Geral
Este repositório contém o website (Landing Page e páginas legais) do aplicativo **How2Burn** (Your AI Fitness Companion). Trata-se de um site estático com foco em apresentação.

## Tecnologias e Linguagens
- **HTML5:** Estruturação semântica do conteúdo (`index.html`, `privacy.html`, `terms.html`).
- **CSS3 (Vanilla):** Estilização visual, layouts responsivos e micro-animações (`style.css`).

## Arquitetura
- **Web Estático:** A arquitetura atual não utiliza frameworks complexos (como React ou Angular), priorizando carregamento rápido, SEO e simplicidade de hospedagem.

*Nota: O perfil do Agente foca em iOS/watchOS avançado para o app principal, porém as tecnologias documentadas aqui refletem o escopo deste repositório web específico.*
